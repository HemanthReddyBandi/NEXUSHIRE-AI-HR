# AI Mock Interview Platform

This is a mock interview platform powered by **OpenAI ChatGPT API** with Web Speech API for voice-over.

## Features

- **AI-Generated Questions**: Uses ChatGPT to generate role-specific interview questions based on resume
- **Voice Over**: Browser's Web Speech API reads questions aloud
- **Speech Recognition**: Records your answers using browser's Speech Recognition API
- **AI Evaluation**: ChatGPT evaluates answers and provides detailed feedback with scores
- **Interview History**: Saves interview records locally
- **Role-Based Questions**: Questions tailored to the selected job role and candidate's resume

## Quick Start

### Prerequisites
- Node.js (v18+)
- OpenAI API key (get free at https://platform.openai.com)

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Get OpenAI API Key

1. Go to https://platform.openai.com/api-keys
2. Sign up (free account with $5 credit for 3 months)
3. Create a new secret key
4. Copy the key

### Step 3: Configure API Key

Create or edit `.env` file in project root:
```
VITE_OPENAI_API_KEY=sk-your-api-key-here
```

Replace `sk-your-api-key-here` with your actual OpenAI API key.

### Step 4: Start the App
```bash
npm run dev
```

Open http://localhost:3000 in your browser and start interviewing!

## How It Works

1. **Enter Job Role & Resume**: Provide context for question generation
2. **Enable Permissions**: Allow camera and microphone access
3. **Interview Starts**: 
   - ChatGPT generates interview questions tailored to your role
   - Browser reads questions aloud using Web Speech API
   - Microphone records your answers
4. **Evaluation**: ChatGPT evaluates your performance and provides feedback
5. **View Report**: See scores, feedback, and improvement suggestions

## Architecture

```
Frontend (React)
    ↓
chatgptService.ts (OpenAI API calls)
    ↓
OpenAI API (https://api.openai.com)
    ↓
ChatGPT Model (gpt-3.5-turbo)
```

- **Questions**: Generated by ChatGPT based on role + resume
- **Voice Over**: Web Speech API (no external service)
- **Evaluation**: ChatGPT analyzes transcript and scores answers
- **Storage**: All interview history saved locally in browser

## Troubleshooting

### "No API key configured"
- Create `.env` file in project root
- Add your OpenAI API key: `VITE_OPENAI_API_KEY=sk-...`
- Restart dev server: `npm run dev`

### "Invalid API key"
- Go to https://platform.openai.com/api-keys
- Check your key is correct
- Ensure you don't have leading/trailing spaces

### "Rate limit exceeded"
- Wait a few minutes or upgrade to paid plan
- Free tier has rate limits

### "Connection timeout"
- Check your internet connection
- Ensure OpenAI service is up (https://status.openai.com)

### Text-to-speech not working
- Requires HTTPS in production (works on localhost)
- Check browser allows microphone/speakers
- Try Chrome, Edge, or Safari

## FAQ

**Q: Is this free?**
A: Free tier gets $5 credit. Most users stay within free tier limits. Pay-as-you-go after that (~$0.002 per interview).

**Q: Can I use my own LLM?**
A: Yes! We also support Ollama (local, free). See `ollamaService.ts` and switch imports in `useInterview.ts`.

**Q: Is my data private?**
A: Interview transcripts are sent to OpenAI API for processing. See their privacy policy: https://openai.com/privacy

**Q: What if I don't have an API key?**
A: The app will use mock questions so you can still test the UI. For real questions, you need an OpenAI account.

**Q: Can I use GPT-4?**
A: Yes! Edit `chatgptService.ts` and change `model: "gpt-3.5-turbo"` to `model: "gpt-4"`. Note: GPT-4 costs more.

## Customization

### Use Different ChatGPT Model
Edit `services/chatgptService.ts`:
```typescript
model: "gpt-3.5-turbo"  // Change to "gpt-4" or other models
```

### Adjust Number of Questions
Edit `constants.ts`:
```typescript
export const MAX_QUESTIONS = 3;  // Change to desired number
```

### Switch to Local LLM (Ollama)
In `hooks/useInterview.ts`:
```typescript
import { generateQuestions, generateEvaluation, textToSpeech } from '../services/ollamaService';
```
